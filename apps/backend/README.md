## Description

Backend for the project

## Project setup

```bash
cp apps/backend/.env.example apps/backend/.env
# note: fill the created .env
npm install
npm run build:dtos
docker-compose up -d
npx prisma migrate dev
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Migrations

```bash
# prepare migration by editing of apps/backend/prisma/schema.prisma
npx prisma migrate dev --name your_migration
```
